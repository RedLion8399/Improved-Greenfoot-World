# Improved Greenfoot World

This is an template for the platform Greenfoot to develop various scenarios based on it.
As the title suggests this is an improvement of an existing programm we use in computer sience class.
In order to simplify programming I refactored it and implemented a few new methods wich do the job way more efficient.

This is not designet to solve a specific problem rather than be universal for multiple scenarios. To fullfill a certain task this template can be used by adding functionality based on the pre-implemented methods.

***

A simple scenario inspired by Karel The Robot and modified by Thomas Kempe & David Tepaﬂe

In diesem Szenario können verschiedene Roboter (Unterklassen von Roboter) erzeugt werden.
Mit Hilfe einiger Sensoren können diese Roboter auch komplexe Aufgaben erledigen.
Die Roboter können zudem Akkus aufnehmen und Schrauben ablegen.
